# Правила разработки для проекта 1С:Предприятие

Этот репозиторий содержит правила и соглашения о коде для разработки на платформе 1С:Предприятие, оформленные в формате MDC (Markdown Context) для использования в Cursor IDE.

## Структура правил

Правила разделены на логические модули в каталоге `.cursor/rules/`:

- **`1c-standards.mdc`** - Основные соглашения о коде, стиль, именование, структура модулей
- **`1c-queries-collections.mdc`** - Работа с запросами, циклами и универсальными коллекциями значений
- **`1c-client-server.mdc`** - Клиент-серверное взаимодействие
- **`1c-errors-security.mdc`** - Обработка ошибок, безопасность и производительность

## Использование

### Для Cursor IDE

1. Скопируйте содержимое каталога `.cursor/rules/` в ваш проект
2. Правила автоматически применятся при работе с файлами `.bsl` и `.os`

### Настройка приоритетов

В файлах правил можно настроить:
- `alwaysApply: true/false` - применять всегда или только при необходимости
- `priority: 1-4` - приоритет применения правил
- `globs` - паттерны файлов, к которым применяются правила

## Дополнительные файлы

- **`CODING_STANDARDS_1C.md`** - Подробная документация по соглашениям о коде
- **`.editorconfig`** - Настройки форматирования для различных редакторов
- **`.vscode/settings.json`** - Настройки редактора для проекта 1С
