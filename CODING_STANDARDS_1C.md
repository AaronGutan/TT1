## Общие принципы

- **Читаемость превыше всего** - код должен быть понятен без дополнительных пояснений
- **Осмысленные имена** - переменные, процедуры и функции должны иметь понятные названия
- **DRY принцип** - не повторяйте код, выносите общую логику в функции
- **Комментируйте сложную логику** - бизнес-правила должны быть задокументированы

## Стиль кода

### Отступы
- Используйте **4 пробела** для отступов (стандарт 1С)
- Не используйте табуляцию

### Длина строки
- Максимальная длина строки: **120 символов**
- Переносите длинные строки с отступом на следующем уровне

### Пустые строки
- Разделяйте процедуры и функции одной пустой строкой
- Группируйте логические блоки кода пустыми строками

### Кавычки
- Используйте **одинарные кавычки** для строковых литералов: `'Текст'`

## Именование

### Переменные
```
✅ Правильно:
ИмяПеременной
Счетчик
МассивДанных
СуммаДокумента

❌ Неправильно:
имя_переменной
i, j, k
temp, tmp
```

### Параметры процедур и функций
```
✅ Правильно:
Процедура ОбработатьДокумент(Документ, ПараметрОбработки)
```

### Приватные переменные модуля
Используйте префикс `м` для модульных переменных:
```
мИмяПеременной
мМассивДанных
мСтруктураПараметров
```

### Процедуры и функции
Начинайте с глагола:
```
✅ Правильно:
ПолучитьДанные()
СоздатьДокумент()
ОбновитьЗапись()
ПроверитьНаличие()

❌ Неправильно:
Данные()
Документ()
Запись()
```

### Константы
Используйте ПРОПИСНЫЕ_СИМВОЛЫ:
```
✅ Правильно:
МАКСИМАЛЬНОЕ_КОЛИЧЕСТВО
ТИП_ДОКУМЕНТА
СТАТУС_ОБРАБОТАН

мКОНСТАНТА_ЗНАЧЕНИЕ
```

Всегда обрабатывайте ошибки:

```bsl

	Сообщение.Текст = "Ошибка при записи документа: " + ОписаниеОшибки();
	Сообщение.Сообщить();

		"Ошибки",
```

## Примеры кода

### Хороший код
```bsl
// Получает список документов за период
//
// Параметры:
//  НачалоПериода - Дата - начало периода
//  КонецПериода - Дата - конец периода
//
// Возвращаемое значение:
//  Массив - массив документов
//
Функция ПолучитьДокументыЗаПериод(НачалоПериода, КонецПериода)

	Если НЕ ЗначениеЗаполнено(НачалоПериода) ИЛИ НЕ ЗначениеЗаполнено(КонецПериода) Тогда
		ВызватьИсключение "Не указан период";
	КонецЕсли;

	Запрос = Новый Запрос;
	Запрос.Текст =
	"ВЫБРАТЬ
	|	РеализацияТоваровУслуг.Ссылка КАК Ссылка
	|ИЗ
	|	Документ.РеализацияТоваровУслуг КАК РеализацияТоваровУслуг
	|ГДЕ
	|	РеализацияТоваровУслуг.Дата МЕЖДУ &НачалоПериода И &КонецПериода";

	Запрос.УстановитьПараметр("НачалоПериода", НачалоПериода);
	Запрос.УстановитьПараметр("КонецПериода", КонецПериода);

	РезультатЗапроса = Запрос.Выполнить();

	МассивДокументов = Новый Массив;
	ВыборкаДетальныеЗаписи = РезультатЗапроса.Выбрать();

	Пока ВыборкаДетальныеЗаписи.Следующий() Цикл
		МассивДокументов.Добавить(ВыборкаДетальныеЗаписи.Ссылка);
	КонецЦикла;

	Возврат МассивДокументов;

КонецФункции
```
